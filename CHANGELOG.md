# Change Log for OXID eShop Module Template

All notable changes to this project will be documented in this file.
The format is based on [Keep a Changelog](http://keepachangelog.com/)
and this project adheres to [Semantic Versioning](http://semver.org/).

## [1.0.0] - Unreleased

### Added
- Github actions Workflows improved
  - Development workflow doesnt save coverage artifacts anymore
  - Manual coverage trigger with saving html coverage directory as artifact
  - Manual shop test with module installed runner
- Improved the way migrations are running. A check if database is not up to date was added.

## [1.0.0-rc.1] - 2022-05-05

### Added
- First version of our reusable examples
